\# Global and Local Variables in R



\## Global Variable

A variable created outside a function.  

It can be used anywhere in the program.



\## Local Variable

A variable created inside a function.  

It can be used only inside that function.



\## Key Difference



| Global Variable | Local Variable |

|----------------|---------------|

| Created outside function | Created inside function |

| Used anywhere | Used only inside function |



\## What I Learned



\- Global variables are accessible everywhere.

\- Local variables work only inside functions.

\- Functions can have their own variables.
