\# Day 6 - Conditional Statements and Loops in R



Today I learned how to control program flow and repeat tasks using loops in R.



\## Topics Covered



\## 1. if Statement

Used to check a condition.



Example:

If age is 18 or above, print eligible to vote.



\## 2. if else Statement

Used when there are two possible outcomes.



Example:

If marks are above 50 = Pass

Else = Fail



\## 3. else if Statement

Used for multiple conditions.



Example:

Assign grades based on score.



\## 4. for Loop

Used to repeat code a fixed number of times.



Example:

Print numbers from 1 to 5.



\## 5. while Loop

Runs code while condition is true.



Example:

Print numbers from 1 to 5 using while loop.



\## 6. break Statement

Stops loop immediately.



\## 7. next Statement

Skips one iteration in loop.



\## Practice Programs



\- Multiplication table of 7

\- Even numbers from 1 to 10

\- Sum of first 5 numbers

\- Decision making in R
